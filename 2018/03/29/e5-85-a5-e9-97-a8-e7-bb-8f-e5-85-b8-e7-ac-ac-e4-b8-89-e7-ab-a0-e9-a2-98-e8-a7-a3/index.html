uva1587 Box 题意：给6个矩形的长和宽，判断他们能否构成长方体。 思路：能构成长方体，长和宽必然最多只有3个不同数，假设其为a,b,c。关系为a<=b<=c，那么我们对这6个矩形排序（假设长<=宽）就能得到ab,ab,ac,ac,bc,bc。不满足的都是不能构成长方体的。 代码：

bool operator < (const Rec& rec) const {
return w < rec.w || (w == rec.w && h < rec.h);
}

bool operator == (const Rec& rec) const {
return w == rec.w && h == rec.h;
}
